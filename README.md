# Portfólio — Victor Nadoti

Portfólio profissional de Victor Nadoti, desenvolvedor Front-End especializado
em sites e landing pages responsivas, autorais e focadas em conversão.

## Tecnologias

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Vercel

## Desenvolvimento local

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Build de produção

```bash
npm run build
npm start
```

## Estrutura principal

- `app/page.tsx`: conteúdo e interações da página
- `app/globals.css`: identidade visual e responsividade
- `app/layout.tsx`: metadados, fontes e configurações gerais
- `public/`: favicon, fotografias e imagens dos projetos

## Publicação

O projeto está conectado à Vercel. Novos commits enviados para a branch `main`
geram uma nova publicação automaticamente.
